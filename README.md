# Construction Management App

This is a React application built with Vite, TypeScript, and Tailwind CSS.

## Prerequisites

- Node.js installed on your machine.
- NPM (comes with Node.js).

## Setup & Run

1. Open your terminal in this directory.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open the link shown in the terminal (usually `http://localhost:5173`) in your browser.

## Project Structure

- `src/components`: Reusable UI components.
- `src/modules`: Feature modules (Dashboard, Labor, Land, etc.).
- `src/utils`: Helper functions.
- `src/types`: TypeScript interfaces.
